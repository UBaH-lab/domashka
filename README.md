Учебный проект по Python
Фича для личного кабинета клиента банка

Инструкция по установке и запуску проекта
1. Клонирование репозитория (если есть)
Если проект хранится на GitHub или другом репозитории:

bash

git clone <URL_репозитория>
cd <имя_проекта>
Или, если проект уже есть локально — просто перейдите в папку проекта.

2. Установка Python и необходимых инструментов
Убедитесь, что установлена версия Python >= 3.13.
Убедитесь, что установлен менеджер пакетов pip.
3. Установка зависимостей
Через Poetry (если используется pyproject.toml)
Установите Poetry (если ещё не установлен):
bash

pip install poetry
Или следуйте официальной инструкции на сайте: https://python-poetry.org/docs/#installation
Установите зависимости и виртуальное окружение:
bash

poetry install
Активируйте виртуальную среду:
bash

poetry shell
Или запускайте команды через poetry run.

Через pip (если есть requirements.txt)
Если в проекте есть файл requirements.txt, установите зависимости командой:

bash

pip install -r requirements.txt
4. Запуск проекта
Обычно проект запускается командой:

bash

python main.py
Или через Poetry:

bash

poetry run python main.py
Если в документации указаны иные команды — используйте их.

Модуль processing
В модуле реализована функция filter_by_state, которая принимает список словарей и опционально значение для ключа state.
Функция возвращает новый список словарей, где значение state совпадает с заданным.
Модуль masks
Функция get_mask_card_number
Принимает номер карты, возвращает замаскированный номер, показывая только первые 6 и последние 4 цифры.
Функция get_mask_account
Принимает номер счёта, возвращает замаскированный вид — видны только последние 4 цифры, перед ними две звёздочки.
Модуль widget
Функция mask_account_card
Принимает строку с типом и номером карты или счёта.
Возвращает строку с правильной маской в зависимости от типа (карта или счёт).
Тестирование
В проекте есть автоматические тесты, которые проверяют функции на различные кейсы.
Тесты запускаются из блока:
python

if __name__ == "__main__":
Как использовать тесты
Убедитесь, что установлен pytest:
bash

pip install pytest
Запустите тесты:
bash

pytest test_module.py
Тесты покрывают:

базовые сценарии;
граничные случаи;
устойчивость к некорректным данным.
