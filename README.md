#учебный проект по Python

   Фича для личного кабинета клиента банка 

Инструкция по установке и запуску проекта
1. Клонирование репозитория (если есть)
Если ваш проект хранится на GitHub или другом репозитории:

bash

git clone <URL_репозитория>
cd <имя_проекта>
или, если у вас уже есть локально — перейдите в папку проекта.

2. Установка Python и необходимых инструментов
Убедитесь, что у вас установлена версия Python >= 3.13.
Установите менеджер пакетов pip, если он ещё не установлен.
3. Установка зависимостей
Через Poetry (если проект использует pyproject.toml)
Установите Poetry (если ещё не установлен):
bash

pip install poetry
или следуйте инструкциям на сайте https://python-poetry.org/docs/#installation

Установите зависимости и виртуальное окружение:
bash

poetry install
Активируйте виртуальную среду:
bash

poetry shell
или выполните команды, вызывающие прямо из этого окружения.

Через pip (если проект использует requirements.txt или что-то подобное)
Если есть файл зависимостей, например, requirements.txt:

bash

pip install -r requirements.txt
4. Запуск проекта
Обычно:
Для запуска: выполните команду, которая запускает основной скрипт проекта. Например:
bash

python main.py
или, если есть команда внутри проекта:

bash

poetry run python main.py
или используйте имя модуля/скрипта, указанное в документации.

МОДУЛЬ PROCESSING
 
В модуле processing реализована функция filter_by_state, которая принимает список словарей и опционально значение для ключа state

Функция возвращает новый список словарей, содержащий только те словари, у которых ключ state соответствует указанному значению.

МОДУЛЬ MASKS

В модуле masks реализованы функции :
get_mask_card_number, которая принимает на вход номер карты и возвращает ее маску. Номер карты замаскирован,
    отображается только первые 6 и последние 4 цифры 
и 
get_musk_account, которая принимает на вход номер счёта и возвращает его замаскированный вид.
        Видны только последние 4 цифры номера, а перед ними-две звёздочки

МОДУЛЬ WIDGET

В модуле widget
 создана функция mask_account_card, которая умеет обрабатывать информацию как о картах, так и о счетах.
Функция принимает один аргумент — строку, содержащую тип и номер карты или счета.
Возвращает строку с замаскированным номером.
